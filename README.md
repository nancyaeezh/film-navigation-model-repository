# film-navigation-model-repository
本仓库专注于影视类网站在导航模型资料仓体系与页面层级方面的公开资料整理，通过文字方式记录分类路径规则、标签组织结构与内链布局模式，建立长期更新的研究档案。项目不涉及影视播放资源，仅归档公开页面信息。内容重点分析页面路径设计与栏目层级关系，用于跨站点结构对比与观察。所有资料来源于可公开访问页面，仅供信息索引与结构研究参考。
